# SereneVoice Bot

Telegram bot converting short text (≤800 chars) into calm, neutral female-voice audio (OGG default, MP3 optional) and returning the file in-chat with 30-day retention.

Spec: [`docs/blueprint.md`](docs/blueprint.md).

Built on [agnt-gm.ai](https://agnt-gm.ai). The whole bot is built and refined here as pull requests across successive build passes.
